# Career Advisor AI
### Built by Dante Beckford | Started April 2026

## What is this?
A python application that asks users about themselves and gathers relevant information such as their goals, situation, and background and uses that information to generate a fully personalized career roadmap using local AI, built from scratch as my first ever project with zero coding prior experience.

## Why I built it
I wanted to create a tool to help other high school students as I was at one point find out how to get to their career, I always knew I wanted to be a software engineer but never quite had the idea of how to get there, of course I knew to go to college, get the major, but we all know that isn't enough in a competitive field, you need to know the right people to contact, how to network, good and free or paid for resources to utilize, and how to practice in your freetime, all things which this bot helps you with, it even helps you find people that can help you along your path, and the right target university to go to and how to access their financial aid systems for those that may not be able to afford it.

## What it does
It collects user information through a conversational interface.
It then sends that information to a local AI model.
And lastly generates a unique personalized career roadmap for any goal.

## Tech used
Python
Ollama (Local AI, can't afford claude and since this is a project that'll likely never have a full launch and will likely be just something I have in my portfolio, will likely never have claude.)

## How to run it
1. Install python from https://www.python.org/
2. Install ollama from https://ollama.com/
3. Run: ollama pull llama3.2
4. Run: pip install ollama
5. Run: python careeradvisorv2.py
